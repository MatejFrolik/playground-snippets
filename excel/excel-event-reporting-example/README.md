# Excel RM Visit – Reporting Playground Project

My **LINKEDIN** [HERE](https://www.linkedin.com/in/matěj-frol%C3%ADk-183812230/).

## Overview
This project demonstrates a simple Excel-based reporting workflow focused on summarizing and visualizing event-related data for management-level review.

The project is intended as a **learning and demonstration example only** and does not represent a production or client-facing solution.

---

## Project Goal
The goal of this project is to present a clear and concise overview answering basic management questions using tabular reports and visualizations in Microsoft Excel.

---

## Data Disclaimer
All data used in this project are **synthetic and anonymized**.  
They are either:
- fully fictional, or
- heavily modified to remove any real-world business meaning.

No real CRM, client, or company data are included.

---

## Project Workflow
1. Data extraction from a CRM-like source (simulated dataset).
2. Data cleaning and table joins using **Power Query**.
3. Final data preparation in **Microsoft Excel**:
   - XLOOKUP / VLOOKUP
   - LET
   - INDEX
   - basic calculations and transformations
4. Visualization using simple charts and summary tables.

---

## Tools Used
- Microsoft Excel
- Power Query
- Basic Excel formulas and functions

---

## Notes
- This project is part of a **personal learning playground**.
- The focus is on understanding data preparation and reporting logic rather than advanced analytics.
- The solution prioritizes clarity and usability over complexity.
